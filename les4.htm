Les 4

In les 4 hebben we alle kennis die we hebben opgedaan in les 3 nodig gehad om onze eigen website te maken. We hebben het programma HTML-kit gebruikt om onze code in te schrijven die benodigd was om het formulier op te stellen. Het maken van het formulier ging eerst niet erg vlot, maar we hebben vragen kunnen stellen die het veel duidelijker maakten. We hebben in onze website ook feedback moeten schrijven over wat we van het project vonden en een korte samenvatting over wat we in de verschillende lessen hebben gezien. Verder hebben we ook ons programma dat we hebben geschreven voor onze Dramco Uno moeten implementeren in onze website. bij ons hield dit programma in dat we wilden tellen hoe vaak iemand per dag zijn koelkast open doet. Deze informatie kan de persoon dan zien op zijn/haar smartphone. Dit kan gebruikt worden door mensen die klimaat bewuster willen leven om te kijken wanneer ze hun koelkast eigenlijk onnodig open doen, of deze te lang laten openstaan. Het programma zou ook een melding kunnen geven als de koelkast niet goed gesloten is en zo veel koude verliest. We moesten ons ook in geringe mate bezig houden met de opmaak van onze website. De code voor de opmaak moesten we schrijven in een linked stylesheet waarbij we als webstandaard CSS moesten gebruiken. We moesten een body, h1 en een paragraaf allemaal een andere opmaak geven. Hiervoor hebben we het programma Topstyle gebruikt dat ons vorige les is uitgelegd. We konden zo eenvoudig en snel al zien hoe onze opmaak er zou uit zien, voor we deze op onze website moesten implementeren.
