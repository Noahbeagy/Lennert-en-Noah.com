Les 3

In les 3 hebben we kennis gemaakt met de webstandaarden HTML en CSS. Deze vormen de basis voor het maken van onze eigen website. HTML staat voor Hypertext Markup Language dit is een client-side webstandaard die gebruikt wordt als taal om documenten op te maken en te linken met andere documenten. We hebben anderzijds ook CSS bekeken, dit staat voor Cascading Style Sheet. Dit is ook een client-side webstandaard. CSS werkt met regels die bepalen hoe een element in een HTML document moet weergegeven worden. Hierdoor kunnen we bijvoorbeeld verschillende lettertypes, achtergrondkleuren ... integreren in 1 pagina. Om meer functionaliteit te kunnen toevoegen aan onze webpagina hebben we verder ook PHP gezien. Dit kan gebruikt worden om pagina's dynamisch te genereren. PHP is in tegenstelling tot HTML en CSS een server-side scripting programmeertaal. Het was makkelijk om code van deze programmeertaal te combineren met HTML. We hebben verder ook PHP gebruikt om ons formulier in de opdracht voor het kamp van de jeugdbeweging op te stellen. Er is ons ook nog het programma Topstyle getoond. Dit programma maakt het vrij eenvoudig om de opmaak van de website snel visueel al eens te tonen. Zo weet je bijvoorbeeld als je een lettertype selecteert onmiddellijk hoe het eruit ziet.
